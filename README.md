# 英雄角⾊管理

**项⽬框架:**
+ SpringBoot+SpringMVC+Spring+Mybatis+Thymeleaf+Bootstrap 

**数据存储:** 
* 默认: H2

**实现功能:** 
+ 1.RESTful 实现CRUD操作 
+ 2.Druid连接池对sql的监控  
+ 3.集成了了Swagger2


**程序运行后查看地址**
+ 首页: http://localhost
+ Swagger2: http://localhost/swagger-ui.html
+ Druid(admin:1234): http://localhost/druid/login.html
